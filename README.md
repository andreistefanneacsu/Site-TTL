# TTL Website

Acest repository conține codul sursă al site-ului oficial al aplicației **TTL**, o platformă de simulare a rețelelor de calculatoare dezvoltată pentru a facilita înțelegerea conceptelor de networking într-un mod interactiv și accesibil.

Website-ul are rolul de a prezenta aplicația, funcționalitățile sale principale și beneficiile pe care le oferă utilizatorilor. Prin intermediul acestuia, vizitatorii pot descoperi informații despre proiect, modul de funcționare al simulatorului și obiectivele urmărite în dezvoltarea platformei.

TTL permite simularea și vizualizarea diferitelor topologii de rețea, oferind un mediu virtual în care utilizatorii pot experimenta și învăța fără a avea nevoie de echipamente fizice. Aplicația este concepută atât pentru studenți și persoane aflate la început de drum în domeniul rețelisticii, cât și pentru utilizatori care doresc să își aprofundeze cunoștințele prin exerciții practice.

Acest site reprezintă punctul central de prezentare al proiectului TTL și oferă o imagine de ansamblu asupra viziunii, funcționalităților și evoluției aplicației.
